# Документация проекта "Демонстрационный стенд OpenPyXL"

## Структура проекта

```
openpyxl-demo/
├── core.py          # Модели данных и генерация примеров (Employee, Sale)
├── utils.py         # Утилиты: стилизация ячеек, форматированный вывод, работа с файлами
├── logic.py         # Логика демонстраций (создание, чтение, формулы, диаграммы и т.д.)
├── app.py           # Главное приложение с меню
└── README.md        # Документация проекта
```

## Описание модулей

### 1. `core.py` — ядро системы

Содержит классы данных и функции для генерации тестовых примеров.

**Модели данных:**
- **`Employee`** — сотрудник (`id`, `name`, `department`, `salary`, `hire_date`)
- **`Sale`** — продажа (`id`, `product`, `quantity`, `price`, `date`)

**Генераторы:**
- **`generate_sample_employees(count=5)`** — создаёт список сотрудников со случайными данными
- **`generate_sample_sales(count=8)`** — создаёт список продаж

### 2. `utils.py` — утилиты и вспомогательные функции

Содержит функции для форматированного вывода и работы с Excel:

- **`print_header(text)`**, **`print_success(text)`**, **`print_error(text)`**, **`print_info(text)`** — цветной/эмодзи-вывод в консоль
- **`apply_header_style(worksheet, cell_range)`** — применяет стиль заголовка (жирный, синий фон, белый текст)
- **`apply_borders(worksheet, cell_range)`** — добавляет тонкие границы к диапазону ячеек
- **`auto_adjust_column_width(worksheet)`** — автоматически подгоняет ширину колонок под содержимое
- **`safe_remove_file(filename)`** — удаляет файл, если он существует, с информированием пользователя

### 3. `logic.py` — логика демонстраций

Каждая функция соответствует пункту меню и демонстрирует определённую возможность `openpyxl`:

- **`demo_create_simple_table()`** — создание нового Excel-файла с листом "Сотрудники", запись данных, стилизация заголовков, границы, автоподбор ширины.
- **`demo_read_data()`** — чтение существующего файла, вывод первых строк и количества записей.
- **`demo_append_data()`** — добавление нового сотрудника в конец таблицы с обновлением границ.
- **`demo_formulas()`** — добавление строк с формулами `SUM` и `AVERAGE` для колонки зарплат.
- **`demo_formatting()`** — условное форматирование (зелёный для зарплат >100k, красный для <70k), курсив для ФИО, выравнивание по центру.
- **`demo_chart()`** — создание столбчатой диаграммы на основе зарплат сотрудников.
- **`demo_multiple_sheets()`** — создание/обновление листа "Продажи" с данными о продажах и вычислением итоговой суммы.

### 4. `app.py` — главное приложение

Содержит:
- **`show_menu()`** — отображение интерактивного меню с пунктами 0–7
- **`main()`** — главный цикл обработки выбора пользователя и вызова соответствующих демонстраций

## Как использовать

### Установка и запуск

1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Установите библиотеку OpenPyXL:
   ```bash
   pip install openpyxl
   ```
3. Скачайте все файлы проекта в одну директорию.
4. Запустите приложение:
   ```bash
   python app.py
   ```

## Демонстрации

### 1. Создание простой таблицы
- Генерируется 5 случайных сотрудников.
- Создаётся файл `demo_workbook.xlsx` с листом "Сотрудники".
- Применяются заголовки, границы, автоширина.
- Выводится содержимое в консоль.

### 2. Чтение данных из Excel
- Открывается созданный ранее файл.
- Выводится название активного листа.
- Печатаются первые 5 строк и общее количество записей.

### 3. Добавление новых данных
- Генерируется ещё один сотрудник.
- Данные добавляются в конец таблицы.
- Обновляются границы и ширина колонок.

### 4. Работа с формулами
- В двух строках после данных добавляются формулы `SUM` и `AVERAGE` для колонки зарплат.
- Строки форматируются жирным шрифтом.

### 5. Форматирование ячеек
- Зарплаты выше 100 000 помечаются зелёным фоном, ниже 70 000 — красным.
- ФИО выделяются курсивом.
- Все ячейки выравниваются по центру.

### 6. Создание диаграммы
- На основе данных о зарплатах строится столбчатая диаграмма.
- Диаграмма размещается на листе справа от таблицы.

### 7. Работа с несколькими листами
- Создаётся (или обновляется) лист "Продажи".
- Добавляются 5 случайных продаж с автоматическим расчётом суммы (`Кол-во * Цена`).
- Применяются заголовки, границы, автоширина.

## Ключевые концепции OpenPyXL

### 1. Создание рабочей книги и листов
- **`Workbook()`** — создать новую книгу.
- **`wb.create_sheet("Имя")`** — добавить новый лист.
- **`wb.active`** — получить активный лист.

### 2. Запись данных
- **`ws.append([...])`** — добавить строку в конец листа.
- **`ws[cell] = value`** — запись в конкретную ячейку.

### 3. Чтение данных
- **`ws.iter_rows(values_only=True)`** — итерация по строкам с получением значений.
- **`ws.max_row`**, **`ws.max_column`** — получение размеров заполненной области.

### 4. Стилизация
- **`Font`**, **`PatternFill`**, **`Alignment`**, **`Border`**, **`Side`** — классы для настройки внешнего вида ячеек.
- Применение стилей через атрибуты ячейки: `cell.font =`, `cell.fill =` и т.д.

### 5. Формулы
- Запись строки, начинающейся с `=`, например `"=SUM(D2:D10)"`. OpenPyXL автоматически интерпретирует её как формулу.

### 6. Диаграммы
- **`BarChart`**, **`Reference`** — создание ссылок на диапазоны данных.
- **`chart.add_data()`**, **`chart.set_categories()`** — добавление данных и подписей.
- **`ws.add_chart(chart, anchor)`** — размещение диаграммы на листе.

### 7. Управление шириной колонок
- **`ws.column_dimensions[letter].width = value`** — установка ширины.

### 8. Работа с несколькими листами
- **`wb.sheetnames`** — список имён листов.
- **`wb["Имя"]`** — доступ к листу по имени.

## Заключение

Данный проект служит наглядным пособием для понимания практической пользы библиотеки `openpyxl`. Каждая демонстрация решает конкретную задачу, возникающую при работе с Excel-файлами в Python: создание отчётов, добавление данных, анализ с помощью формул, визуализация диаграммами и улучшение читаемости с помощью форматирования.

Библиотека OpenPyXL особенно полезна для:
- **Автоматизации отчётности** — генерация Excel-файлов с данными из БД или API.
- **Обработки данных** — чтение, анализ и модификация существующих таблиц.
- **Создания дашбордов** — встраивание диаграмм и формул прямо в Excel.
- **Обмена данными** — экспорт/импорт в удобный для пользователя формат.

Проект демонстрирует, как с помощью минимального количества кода можно создать полноценное приложение для управления Excel-файлами, которое будет работать без ошибок и наглядно показывать все ключевые возможности библиотеки.
